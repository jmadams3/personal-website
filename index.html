This was a collaborative project I did for HackNC. Program scans in historical data from a file and produces a line graph of average maximum temperature, average minimum temperature, and average precipitations for each year. Nonograms Puzzle Application

A complete, functioning GUI implementation of the single-player logic puzzle nonograms. Nonograms are picture logic puzzles in which cells in a grid must be colored or left blank according to numbers at the side of the grid to reveal a hidden picture. I used the MVC and Observable Design Patterns to complete this project.

Covid Hub

This was a collaborative project I did for HackTheLib. We created a centralized website that consists of a Home Page, About Us tab, Discussion Forum, Interactive Map of the covid cases, and a Relief Form to reduce the abstraction barrier for people. We used the Maps JavaScript API by Google to create a map and add markers, TrackCorona API to get COVID-19 country data, Firebase to store data submitted in the Relief Form Page, and JQuery to make the bot.
